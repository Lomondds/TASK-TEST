Проектирование.


Данная программа писалась с использованием базы данных SQLITE.  База данных database1.db содержит три таблицы:
Таблица Trips содержит такие данные, как id, название поездки, протяжённость пути, тип транспорта, который с помощью внешнего ключа имеет связь с таблицей TypeVehicle. Эта таблица  имеет id, название транспорта, расход топлива, количество топлива, а также измерения по которому ведётся расчёт затрат топлива. Эта таблица по внешнему ключу имеет связь с таблицей Measurement, она имеет только id и само измерение, выраженное в л/км и кг/ч.


Функционал.


Пользователь может полностью управлять такими таблицами как: Trips и TypeVehicle; кнопками - Добавить, Удалить, Обновить.
Для того, чтобы пользователь мог узнать сколько может пройти километров тот или иной транспорт, ему необходимо указать поездку в сплывающем списке, которую он добавил в таблицу Trips. Программа покажет путь, который необходимо преодолеть, топливо, которое есть у транспорта, также укажет, сколько тот или иной транспорт сможет пройти за это топливо и укажет недостаточного топлива.
